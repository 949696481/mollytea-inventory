下载 Windows 版

下载 →

看到"不安全"或"病毒"提示是正常的，软件没有购买付费签名证书，不是真的有病毒。

遇到提示怎么办？
  1. 浏览器下方出现"不常用的下载"提示 → 点旁边的"..." 选"保留"（或"仍要下载"）
  2. 双击打开时出现"Windows 已保护你的电脑" → 点左下角"更多信息" → 再点"仍要运行"
  3. 如果文件下载后又不见了（被杀毒软件删除）→ 打开"Windows 安全中心" → "病毒和威胁防护" → "保护历史记录"，找到被隔离的文件点"还原"；再进"病毒和威胁防护设置" → "排除项"，把下载文件夹加进去，然后重新下载一次
